### 🚨 問題與解法 (Troubleshooting)
> 🐛 **遇到困難**：Colab 渲染報錯 `Could not find composition with ID CryptoPaymentAnimation`。
> 💡 **解決方案**：確保本地代碼推送到 GitHub (`git push`)，並在 Colab 端執行 `!git pull` 確保雲端代碼與本地同步。
> 🐛 **遇到困難**：中文字在預覽正常，但在 Colab 渲染時呈現為方塊（Tofu 問題）。
> 💡 **解決方案**：
> 1. **系統層級**：在 Colab 筆記本加入 `!sudo apt-get update && sudo apt-get install -y fonts-noto-cjk`，將思源黑體安裝進 Ubuntu 系統底層。
> 2. **專案層級**：簡化 `@remotion/google-fonts` 的加載邏輯，放棄手動指定 `subsets` 避免映射錯誤。
> 3. **測試結果**：成功在雲端渲染出正確中文字體。
