# 2.5D Product Showcase

## 情境
產品截圖展示畫面太過平面，缺乏高級感。

## 解法

### 1. 手機容器包裹
將螢幕截圖放入手機外框容器中：
```tsx
<div style={{
  borderRadius: 40,
  overflow: 'hidden',
  boxShadow: '0 25px 50px rgba(0,0,0,0.25)',
  width: 300,
  height: 600,
}}>
  <Img src={staticFile('screenshot.png')} style={{ width: '100%' }} />
</div>
```

### 2. 3D 旋轉進場
外層加上 3D transform，配合 heavy mass spring 做進場轉動：
```tsx
const rotation = spring({
  frame,
  fps,
  config: { damping: 100, stiffness: 80, mass: 2 }, // Heavy = 沉穩感
});

const rotateY = interpolate(rotation, [0, 1], [25, 0]);
const scale = interpolate(rotation, [0, 1], [0.8, 1]);

<div style={{
  perspective: 1500,
  transform: `rotateY(${rotateY}deg) scale(${scale})`,
}}>
  {/* 手機容器 */}
</div>
```

### 3. SVG 背景線條點綴
在容器後方加入 SVG 線條，用 `stroke-dashoffset` 做動態畫線：
```tsx
const lineProgress = interpolate(frame, [10, 40], [1, 0], {
  extrapolateRight: 'clamp',
});

<svg>
  <line
    x1="0" y1="0" x2="400" y2="400"
    stroke="#D4694A"
    strokeWidth={2}
    strokeDasharray={566}
    strokeDashoffset={566 * lineProgress}
  />
</svg>
```
